# /kiips-linked-approval-template

결재 연계 문서(LinkedApproval) HTML 템플릿과 데이터 바인딩 로직을 생성합니다.

## Instructions

1. `kiips-linked-approval-template` 스킬을 로드하여 가이드라인을 참조하세요.
2. 사용자에게 다음 정보를 확인하세요:
   - **대상 모듈** (FD/AC/IL/IV/PG/RM/SY/DAOL)
   - **doc_id** (결재 연계 문서 ID)
   - **화면 레이아웃** (스크린샷 또는 필드 목록)
   - **데이터 접근 패턴** (VIEW, LIST 등)
3. 스킬의 Step 0~5 절차에 따라 템플릿을 생성하세요.
4. 검증 체크리스트를 모두 통과한 후 완료 보고하세요.
